Summary: Latitude and Temperature Changes

   

We see the relationship between city latitude and max temperature to have a negative parabolic trend. This means that has the latitude increases, the temperature decreases with a decreasing rate. When applying a linear regression to this line, we found the r-square value to be low.

For future analysis, it would be useful to apply a different type of model to better estimate the effect latitude has on maximum temperature.
